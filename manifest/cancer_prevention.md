## 詳細
### 1. がん検診率の向上：先進国の中でも低い日本の検診率を底上げし、早期発見を推進

- **マニフェスト**  
  がん検診受診率を全体で60％超へ引き上げ、早期発見・早期治療による死亡率・医療費を大幅削減

- **政策案**
  - #1 全国統一の「がん検診コール＆リコール制度」導入
    - 住民基本台帳とマイナンバー連動で対象者に個別通知・予約案内を自動化
    - 職域検診とのデータ統合を推進し、「二重カウント」防止と漏れのない仕組みを構築
    - 自治体への運営補助金を拡充し、**参加インセンティブ（交付金）**を付与
      - → 全国民に「年1回（または隔年）のがん検診が当たり前」とする習慣の形成
  - #2 要精密検査者への追跡・支援体制強化
    - 精密検査未受診者フォローアップを義務化  
    - 地域拠点病院・がん相談支援センターと連携した「がん検診ナビゲーター制度（仮）」導入  
    - フォロー率改善に応じた自治体インセンティブ制度（成果連動交付金）の導入
      - → 「見つけただけで終わらない」体制整備
  - #3 がん検診データの国家管理と見える化：透明性と改善を促す情報整備
    - 受診率・部位別結果・精密検査率を全国共通フォーマットで収集  
    - 国主導で **「がん検診登録制度」** を創設し、OECD等と比較可能な評価指標に基づく監査を実施  
    - 自治体・医療機関別に公開ランキング化（ただし偏差値的指標で公平性配慮）  
        - → データドリブンな改善とガバナンス強化
  - #4 デジタル予約・受診リマインドの仕組み整備：利便性で受診率アップ
    - マイナンバーポータルやLINEなどと連携し、検診予約・リマインダー通知・結果管理を一元化  
    - 民間アプリ・健康保険組合と連携し、**予防ポイント制度（例：受診で保険料割引）**を試験導入  
        - → スマホで完結する「検診のUX革命」
  - #5 若年・現役世代への啓発と就労支援連動：「健康なうちにこそ必要」の社会常識化
    - 中学/高校・大学での「がん教育義務化」（20代女性の子宮頸がん受診率向上のため）  
    - 企業との連携で「がん検診義務化＋特別休暇制度」を推進（中小企業には補助金）  
    - メディア・SNSキャンペーンと連動した年次統一検診ウィークの創設  
        - → 「健康なうちにこそ必要」な認識の社会実装
- **現状認識・課題意識**
  1. 組織的スクリーニングの未整備
      - 他国では、住所ベースで検診案内（call‑recall）されており、受診率が高い。一方、日本では職場健診と市町村検診が併存し、統合された仕組みが不十分。
  2. 情報・啓発の不足
      - 「時間がない」「健康なので不要」といった受診しない理由が多く、本当の必要性が伝わっていない 。
  3. 経済的・心理的障壁
      - 安価（あるいは無料）で受けられる自治体検診が知られていない。また、検査痛への不安により受診を躊躇 。
  4. 質管理（フォロー体制）の弱さ
      - 要精密検査者の追跡率に地域差が大きく、検診後のフォローが不十分 。
  5. データ整備の遅れ
      - 国レベルのがん検診登録制度やモニタリングシステムが未完備、受診率や精密検査フォローの全体像が不透明 。

- 背景
  - 日本の平均的ながん検診受診率は約40〜50％とされており、60％以上という国の目標には達していない。
  - 一方、先進各国の平均の検診受診率は概ね60％超～80％に及んでおり、日本とは20〜30ポイントの差がある。
  - 米国ミシガン大学の研究では、1996–2020年の25年間で早期スクリーニングによる労働生産性向上などを含めた経済効果を 6.5兆ドル規模と試算しています（医療費だけでなく、社会的価値も含む）。
  - 国家予算に占める医療費が増大する中、日本においても、早期発見・早期治療による死亡者数の低減と年間ベースでのコスト削減を本格的に取り組む必要があるのではないか

### 2. HPVワクチンの男女接種促進：子宮頸がんをはじめとする関連がんのリスクを減少させる
- **マニフェスト**  
  誰も取り残さないワクチン政策：HPVワクチン未接種世代の数千億円規模の損失抑制とがん罹患の予防

- **政策案**
短期政策案
  - #1 キャッチアップ接種の延長・対象拡大
    - 現在の2026年3月末終了予定の再延長をさらに数年延長し、対象を1995～2008年生まれまで広げる。
    - 住民基本台帳とマイナンバー連動で対象者に個別通知・予約案内を自動化
  - #2 個別通知の強化
    - 住所情報とマイナポータル連携による「多段階リマインド通知（郵送・SMS・LINE等）」の実施。
  - #3 接種機会の拡充
    - 大学の健康診断や企業の職域健診でワクチン接種を併設。
    - 地方自治体に予算措置を設定。
  - #4 SNS・動画による啓発
    - 若者・親世代向けに医師・当事者が出演する動画配信やインフルエンサーとの連携。
  - #5 医療従事者への研修強化
    - 接種相談・副反応対応の体制整備を進め、医療現場で接種不安の払拭。
中長期政策案
  - #1 学校接種制度の段階的導入
    - オーストラリア・英国モデルを参考に、中学校での集団接種のモデル実証→全国展開へ。
  - #2 男子接種の定着と理解促進
    - 男子接種の意義（中咽頭がん・肛門がん予防）を明記した啓発教材・保健教育の強化。
  - #3 HPVワクチンと検診の統合戦略
    - マイナポータル等により、接種有無に応じた検診リマインドや医療機関誘導を個別最適化。
  - #4 ワクチンデータの一元管理
    - 母子手帳・接種履歴をデジタル化し、全国で統一的に追跡・評価可能な体制へ。
  - #5 がん教育の制度化
    - 保健体育の学習指導要領に「性感染症とHPV予防」の明記、予防教育の必修化を目指す。


- **現状認識・課題意識**
  1. 接種機会を逃した「空白世代」の未接種者多数
      - 2013～2021年の勧奨中止により、1997〜2006年生まれを中心に約300万人が未接種のまま放置。
  2. キャッチアップ接種の周知・実施が不十分
      - 2022年～2025年度末までの期間限定措置だが、接種率は想定よりも低調。
  3. 接種への不安・誤情報が依然根強い
      - 副反応報道の記憶が親世代に残り、接種回避・無関心を助長。
  4. 男子接種の認知と実施が進んでいない
      - 2020年度から定期接種対象だが、周知不足と「男子には関係ない」という誤解が根強い。
  5. 医療・教育現場での接種導線が弱い
      - 大学・職場での接種体制が未整備。学校接種も未導入。
  6. 子宮頸がん検診受診率が低い
      - 未接種層にとっては特に重要な検診が、十分に機能していない。
  7. ワクチン格差による将来的な健康・経済格差の拡大
      - 同世代でも接種有無で命や経済的損失に大きな差が出る可能性。

- 背景
  - HPVワクチンの接種機会を失った「空白世代」への対応は、予防医療における公正性の観点から極めて重要。
  - ワクチンを打てた世代と打てなかった世代で、将来のがん罹患リスクに大きな格差が生じることは、健康の不平等を拡大させる深刻な問題となる。
  - さらに、子宮頸がん罹患に伴う医療費や労働損失、育児困難などの社会的・経済的損失は一人あたり数千万円規模とされ、社会全体にも大きな負担となる。
  - 公平な予防医療と長期的な財政健全化の両立のためにも、国家的な救済策が不可欠と考える。
